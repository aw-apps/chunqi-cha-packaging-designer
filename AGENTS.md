# Agent 工作指南

此專案為純前端（GitHub Pages）MVP，目標是讓使用者能：
1) 在 2D 展開圖編輯各面貼圖（上傳/拖曳/縮放/旋轉）
2) 即時看到 3D 折盒預覽
3) 用 localStorage 自動保存，並可 JSON 匯出/匯入

## 原則
- 盡量零相依（不引入外部套件）
- 所有狀態都可序列化（JSON）
- 3D 預覽優先用 CSS 3D（避免 WebGL 依賴）

## 主要檔案
- `index.html`：UI
- `style.css`：佈局與 3D
- `app.js`：狀態、2D 繪製、互動、匯入匯出、3D 貼圖更新

## 驗證
- 能上傳圖片到任一面並調整位置/縮放/旋轉
- 切換面不會丟失狀態
- 重新整理頁面仍保留設計
- 匯出後清除，再匯入可完整還原
- 3D 預覽與 2D 一致
